Timeline

  • November 2017: EasyLEED 2.5 released. UI improvements and fixes, support for pausing runs.
  • October 2017: EasyLEED 2.4 released. Support for INI configuration files, bug fixes.
  • June 2017: EasyLEED 2.3.1 released. Support for Qt5, bug fixes.
  • March 2017: EasyLEED 2.2 released. Python 3 support.
  • July 2015: EasyLEED 2.1 released. Easier installation via setuptools.
  • October 2014: EasyLEED 2.0 released. Many improvements and bug fixes.
  • August 2013: Code and documentation moved to Github.
  • July 2012: Publication of paper on tracking algorithm.
  • July 2010: Start of development.
(see also the changelog)
